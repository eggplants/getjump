getjump

getjump

  • Retrieve and save images from manga distribution sites using GigaViewer
    • If you read retrieved comics as combined PDF, use: eggplants/mkbook

_Note: Redistribution of downloaded image data is prohibited. Please keep it to private use._

  • <host>/(episode|magazine|volume)/<number>
    • e.g. https://shonenjumpplus.com/episode/13932016480028799982

Download all series

To download all series at once:

from getjump import GetJump as g

G = g()
next_uri = "https://shonenjumpplus.com/episode/13932016480028799982.json"
while next_uri:
    next_uri, prev_title, saved = G.get(next_uri, overwrite=False)
    if saved:
        print("saved:", prev_title)
    print("next:", next_uri)

Login

To get purchased or login-required works:

from getjump import GetJump as g

G = g()
G.login("https://shonenjumpplus.com", username="***", password="***")
G.login("https://comic-days.com", username="***", password="***")
...
G.get(...)

267    def __get_image(self, image_src: str, div: int = 4, mul: int = 8) -> Image.Image:
268        img = Image.open(BytesIO(self._session.get(image_src, timeout=10).content))
269        img_width, img_height = img.size
270        fixed_width = int(float(img_width) / (div * mul)) * mul
271        fixed_height = int(float(img_height) / (div * mul)) * mul
272        buff: list[list[Image.Image]] = []
273        for x in range(div):
274            inbuff: list[Image.Image] = []
275            for y in range(div):
276                cropped = img.crop(
277                    box=(
278                        fixed_width * x,
279                        fixed_height * y,
280                        fixed_width * (x + 1),
281                        fixed_height * (y + 1),
282                    ),
283                )
284                inbuff.append(cropped)
285            buff.append(inbuff)
286
287        for y, inbuff in enumerate(buff):
288            for x, cropped in enumerate(inbuff):
289                img.paste(cropped, box=(fixed_width * x, fixed_height * y))
290        return img
