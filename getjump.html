getjump

getjump

  • Retrieve and save images from manga distribution sites using GigaViewer
    • If you read retrieved comics as combined PDF, use: eggplants/mkbook

_Note: Redistribution of downloaded image data is prohibited. Please keep it to private use._

Download all series

To download all series at once:

from getjump import GetJump as g

G = g()
next_uri = "https://shonenjumpplus.com/episode/13932016480028799982.json"
while next_uri:
    next_uri, prev_title, saved = G.get(next_uri, overwrite=False)
    if saved:
        print("saved:", prev_title)
    print("next:", next_uri)

Login

To get purchased or login-required works:

from getjump import GetJump as g

G = g()
G.login("https://shonenjumpplus.com", username="***", password="***")
G.login("https://comic-days.com", username="***", password="***")
...
G.get(...)

270    def __get_image(self, image_src: str, div: int = 4, mul: int = 8) -> Image.Image:
271        img = Image.open(BytesIO(self._session.get(image_src, timeout=10).content))
272        img_width, img_height = img.size
273        fixed_width = int(float(img_width) / (div * mul)) * mul
274        fixed_height = int(float(img_height) / (div * mul)) * mul
275        buff: list[list[Image.Image]] = []
276        for x in range(div):
277            inbuff: list[Image.Image] = []
278            for y in range(div):
279                cropped = img.crop(
280                    box=(
281                        fixed_width * x,
282                        fixed_height * y,
283                        fixed_width * (x + 1),
284                        fixed_height * (y + 1),
285                    ),
286                )
287                inbuff.append(cropped)
288            buff.append(inbuff)
289
290        for y, inbuff in enumerate(buff):
291            for x, cropped in enumerate(inbuff):
292                img.paste(cropped, box=(fixed_width * x, fixed_height * y))
293        return img
